什么是 Claude Code 中转站？

Claude Code 中转站（也称镜像站）是为解决国内用户访问官方 Claude Code 服务困难而设立的代理服务平台。这些平台通过技术手段，为用户提供稳定、快速的 Claude Code API 访问服务。

中转站的主要作用包括：解决网络连接问题、提供本土化支付方式、降低使用成本、提供中文技术支持等，让国内开发者能够更便捷地使用 Claude Code AI 编程助手。

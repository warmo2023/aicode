一、Super Claude Code 是什么？

1.1 核心概念

Super Claude Code（SuperClaude Framework）是托管在 GitHub 的 SuperClaude-Org/SuperClaude_Framework 的开源项目。它通过元编程技术和配置注入,将原生 Claude Code 转变为结构化开发平台。

重要声明： SuperClaude 与 Anthropic 公司无关。Claude Code 由 Anthropic 开发和维护,SuperClaude 只是基于 Claude Code 的第三方增强框架。

二、16 个智能代理系统

2.1 什么是智能代理？

Super Claude Code 提供 16 个领域专家代理,根据上下文自动协调工作。当你运行一个命令时,多个代理会自动接力完成任务,无需手动切换。

自动代理协作示例：实现登录功能 

> /sc:implement user login

[1] Product Manager 分析需求

[2] Frontend Architect 设计 UI
    ✓ 设计登录表单
    ✓ 添加表单验证

[3] Backend Engineer 实现 API
    ✓ POST /api/login 接口
    ✓ JWT Token 生成

[4] Database Designer 设计表结构
    ✓ users 表（id, email, password_hash）

[5] Security Engineer 安全审查
    ✓ 检查 SQL 注入风险 ✓
    ✓ 检查密码加密 ✓

[6] QA Engineer 生成测试
    ✓ 单元测试、集成测试

Claude Code 安装教程（基于 GACCode｜Windows / macOS / Linux 新手版）

发布时间：2025年10月08日 • 阅读时间：约 8 分钟 • 分类：问题解决
GACCode 安装教程 Windows / WSL macOS Linux

文章摘要

本文只讲 基于 GAC 的安装路径：注册 GAC 账号 → 购买或领取兑换码 → 激活后安装 gaccode 镜像版 Claude Code， 首次登录完成验证码校验，必要时用 claude --pick-relay 选择延迟更低的节点。附 Windows（含 WSL）、macOS、Linux 全平台步骤与排障。

一、准备与账号：从 GAC 出发

  1. 注册/登录 GAC 账户：前往 gaccode.com 创建账户或登录。
  2. 先体验或直接购买：想感受一下先去 免费体验；确认使用后到 购买中心 获取兑换码。
  3. 激活兑换码：回到 gaccode.com 账户页绑定兑换码并激活订阅，随后按需在控制台生成密钥或直接走命令行验证码登录。

GAC 提供 镜像版 Claude Code 安装命令 与 节点选择 等能力，适合国内网络环境。

二、安装 Node.js（多平台一步到位）

macOS（推荐 nvm） 

# 安装 Homebrew（已装可跳过）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 nvm 并初始化
brew install nvm
mkdir -p ~/.nvm
echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.zshrc
echo '[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && . "/opt/homebrew/opt/nvm/nvm.sh"' >> ~/.zshrc
source ~/.zshrc

# 安装并启用 Node LTS
nvm install --lts
nvm use --lts
node -v
npm -v

Windows（建议 WSL）

  1. 启用 WSL（Ubuntu）：在 PowerShell（管理员）执行 wsl --install 并重启。
  2. 重启后打开 Ubuntu 终端，执行以下命令安装 Node LTS： 
# 在 Ubuntu 终端执行
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
nvm use --lts
node -v
npm -v

如不使用 WSL，也可安装 Windows 版 Node.js，但更容易遇到路径/权限问题。

Linux（以 Ubuntu 为例） 

# 安装 nvm（官方脚本）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# 重新加载环境并安装 Node LTS
source ~/.bashrc
nvm install --lts
nvm use --lts

# 验证版本
node -v
npm -v

三、安装 Claude Code（GAC 镜像命令）

准备好 Node.js 后，在终端执行下面的一条命令完成安装： 

npm install -g https://gaccode.com/claudecode/install --registry=https://registry.npmmirror.com
claude --version

如果此前装过官方版，先执行：

npm uninstall -g @anthropic-ai/claude-code

镜像安装命令与卸载/重装流程为 GAC 官方给出的路径；国内网络建议带上 --registry 提升成功率。

四、首次启动与登录（验证码方式）

  1. 进入你的项目目录，运行 claude。
  2. 命令行会给出一个链接，按提示完成验证并输入验证码。
  3. 登录成功后，按向导选择主题、是否允许执行 Bash、以及默认目录。

小贴士 · GAC 的密钥/兑换码获取

先去 免费体验 感受下；满意后在 购买中心 获取兑换码，并在 gaccode.com 激活，随后即可正常使用（或在控制台创建 API Key 走 API 路线）。

五、常用命令与节点优化

  • claude -c：恢复上一次中断的工作会话。
  • claude --pick-relay：列出可用节点，↑↓ 选择延迟更低的线路。
  • 清理登录信息（遇到异常验证时）： 
    rm -rf ~/.claude ~/.claudecode ~/.claude.json
  • 升级或彻底重装： 
    # 升级
npm install -g https://gaccode.com/claudecode/install --registry=https://registry.npmmirror.com

# 若升级异常，彻底重装
npm uninstall -g @anthropic-ai/claude-code
rm -rf ~/.claude ~/.claudecode ~/.claude.json
npm install -g https://gaccode.com/claudecode/install --registry=https://registry.npmmirror.com

六、在 VS Code 里使用

  1. 打开 VS Code，进入扩展商店搜索 Claude Code 并安装。
  2. 或按 Ctrl+P/Cmd+P 输入 ext install anthropic.claude-code 快速安装。
  3. 侧边栏将出现 Claude Code 面板，可与 CLI 共享同一登录态。

七、常见问题（FAQ）

无法登录 / OAuth 验证卡住怎么办？

请先确认系统未设置代理变量；如仍失败，忽略自动弹出的浏览器，复制终端中的链接手动验证。必要时清理 ~/.claude* 再重试。

网络不稳定，如何切换更快的线路？

使用 claude --pick-relay 列表中选择延迟更低的节点；这是一条 GAC 推荐的网络优化方式。

如何理解 积分 与扣费？

不同模型、上下文长度会影响积分消耗。可参考 GAC 提供的「Claude Model 成本可视化」示例了解大致消耗区间。

准备好开始写代码了吗？

先免费体验，满意后再购买兑换码激活；遇到安装/网络问题，随时加站内客服微信咨询。

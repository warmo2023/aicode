GAC / GACCode 是什么？

GAC / GACCode 是一体化的开发者订阅服务，面向 AI 编程助手场景，支持 Claude Code 与 OpenAI Codex 两套工具链，积分在两者间通用扣费。 订阅后你可以按需选择两种使用方式：更易上手的“改版客户端”或可紧跟上游的“官方原版+API”。

  • • 支持 Claude Code + Codex 双工具，积分共享
  • • 提供 改版客户端（更省心）与 官方原版+API（更灵活）两种路径
  • • 适配线路、减少复杂网络配置带来的不稳定因素

  3. 激活订阅：回到 gaccode.com 的账户页使用兑换码完成激活。

两种使用方式：改版客户端 vs 官方原版 + API

方式 A：改版 Claude Code 客户端（省心）

面向新手或希望“装完即用”的用户。改版客户端会自动适配线路、降低配置成本，更有利于日常稳定使用。

常见安装命令示例 

npm install -g https://gaccode.com/claudecode/install --registry=https://registry.npmmirror.com

安装后运行 claude，按照提示在浏览器完成登录和绑定。

方式 B：官方原版 + API（灵活）

面向进阶用户或团队。保持与官方版本同步，通过 GACCode 提供的 API Key 接入，按需配置环境变量和 Base URL。

常见安装命令示例 

npm install -g @anthropic-ai/claude-code

配置环境变量并运行 claude 启动使用。

Codex 同步支持：与 Claude Code 共享积分

订阅激活后，你还可以使用 OpenAI Codex。两套工具的使用额度按同一账户积分计费与补充，更便于统一管理。

一键安装示例 

npm install -g https://gaccode.com/codex/install

安装完成后运行 codex，首次启动会自动引导登录、完成 API 配置。
